# Oye, Vaquero! 🚲🛴🛼
### **Descripción**

Oye Vaquero! es una plataforma web enfocada en ofrecer una alternativa de movilidad urbana en Monterrey mediante la renta de bicicletas, scooters y patines.

El objetivo del proyecto es brindar a los usuarios una opción de transporte práctica, accesible y sustentable para desplazarse por la ciudad, reduciendo tiempos de traslado y promoviendo una movilidad más eficiente y amigable con el medio ambiente.

A través de la página, los usuarios pueden explorar las opciones de transporte disponibles, conocer sus características y acceder a una experiencia moderna e intuitiva para facilitar su renta.

## Cómo correr el proyecto

Para ejecutar el proyecto de manera local, sigue estos pasos:

1. FrontEnd

Abre una terminal dentro de la carpeta FrontEnd y ejecuta los siguientes comandos:
```
npm install
npm run dev
```
Esto iniciará el servidor de desarrollo del frontend.

2. BackEnd

Abre una segunda terminal dentro de la carpeta BackEnd y ejecuta:
```
npm install
node server.js
```
Esto levantará el servidor del backend.

## Acceder a la aplicación

Una vez que ambos servidores estén corriendo, abre tu navegador y entra a la siguiente ruta:
```
http://localhost:5173/
```
